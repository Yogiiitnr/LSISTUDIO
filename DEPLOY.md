# LSI Acoustic Studio - Deployment Instructions

## 🚀 Deploy on Streamlit Community Cloud (RECOMMENDED)

**Streamlit Community Cloud is PURPOSE-BUILT for Streamlit apps and is FREE.**

### Steps:

1. **Go to:** https://share.streamlit.io

2. **Click "Deploy an app"**

3. **Connect with GitHub:**
   - Click "Connect your GitHub repo"
   - Sign in with GitHub
   - Select `Yogiiitnr/LSISTUDIO`

4. **Configure Deployment:**
   - **Repository:** `Yogiiitnr/LSISTUDIO`
   - **Branch:** `main`
   - **Main file path:** `app.py`

5. **Click "Deploy"** ✅

**Your app will be live at:** `https://lsistudio.streamlit.app`

---

## Alternative: Railway.app

If Community Cloud doesn't work, try:
1. Go to https://railway.app
2. New Project → Deploy from GitHub
3. Select `Yogiiitnr/LSISTUDIO`
4. Railway auto-detects requirements.txt
5. Click Deploy

---

**No servers to manage. No configuration needed. Just click Deploy!**
